## sentient.vim

Vim syntax highlighting for [Sentient](http://sentient-lang.org/).

## Installation

I'd recommend using [pathogen.vim](https://github.com/tpope/vim-pathogen):

```bash
cd ~/.vim/bundle
git clone https://github.com/sentient-lang/vim-sentient.git
```
